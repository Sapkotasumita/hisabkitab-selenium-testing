# HisabKitab Selenium Testing

This repository contains Selenium automation tests for the *HisabKitab* application.  
The tests are written in Python using Selenium WebDriver.

# Structure
- `login.py` — Script to automate login scenarios.
- `hisabkitab_login_results.csv` — Output results of login tests.
- `hisabkitab_screenshot/` — Screenshots captured during test execution.

# How to Use
Install Python requirements:
pip install -r requirements.txt

Run the test:
python login.py
